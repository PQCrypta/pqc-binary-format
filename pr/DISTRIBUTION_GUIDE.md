## 📧 Email Pitch Template

**Subject:** New Open-Source Solution to Post-Quantum Cryptography Interoperability Problem

**Body:**

Hi [Name],

I wanted to share a significant development in post-quantum cryptography that I think your readers would find valuable.

**The Problem:** Following NIST's 2024 PQC standardization, organizations implementing quantum-resistant encryption are hitting a critical barrier—every implementation uses incompatible data formats. ML-KEM-1024 in Rust can't read ML-KEM-1024 from Python. It's blocking real-world adoption.

**The Solution:** We're open-sourcing PQC Binary Format v1.0—the first universal, standardized binary format for post-quantum encrypted data interchange.

**Why It Matters:**
- Works across 28+ algorithms and all major programming languages
- Enables seamless enterprise PQC migration
- Open source (MIT/Apache-2.0) with active community
- Already published on crates.io with full documentation

**Key Resources:**
- Press release: [Attached]
- GitHub: https://github.com/PQCrypta/pqcrypta-community
- Live demo: https://crates.io/crates/pqc-binary-format
- Technical docs: https://docs.rs/pqc-binary-format

I'm happy to provide additional technical details, arrange interviews, or answer any questions.

Would this be interesting for your coverage?
